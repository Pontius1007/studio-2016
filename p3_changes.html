General changes

While the general changes from design to finished product are not that visible, some changes were made. We decided to streamline the producer page and make some changes to the general layout there. We edited the photoes and decided to put the information under the producer, rather next to it. This is so it will more closely resemble the Lessons page. We have added several pictures and also changed the name on a few. The logo is fictional and found on the internet. Picture will be linked down below. Rest are as described in P2-Design. We had some minor changes to the font-size as described more clearly in style.css. We made the size more responsive relative to the body and tried to aboid using hard-coded px size. In P2 it was described tht the form validation would have something to do with a shop, that was wrong and changed. It now checks that the user has entered a name and email when contacting a producer.

The group decided that a dedicated site map was redundant, considering you can go from anywhere on the web page to anywhere else, just using the navigation bar. The navigation bar in this case therefore works as the site map, making a dedicated site map not necessary. The feedback for navigation on the page from the testing also confirmed that a site map was not necessary.

Our final product is very much the same as described and shown in P2-Design. Therefore the changes will not be as much compared to anyone with a lot of changes.

Modularity

The modularity requirment was met by making a javascript file, header_footer.html, that makes the header and footer for all pages. This script inserts the header and footer into all pages. We decided to also include the navigation bar in this script as well, due to it is the same on all pages. We are also scaling every picture and the layout in general when a user modifies the screen size. This is so the page looks good for every user, not just people using 1920/1080 etc. We set a minimum width so the site will hold its structure when compromised too much.
